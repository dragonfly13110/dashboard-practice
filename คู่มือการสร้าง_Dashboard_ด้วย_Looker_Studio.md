## 📚 บทที่ 0: ทำไมนักส่งเสริมการเกษตรต้องทำ Dashboard?

### 🎯 ปัญหาที่เกษตรตำบลเจอทุกวัน

| ปัญหา | ผลกระทบ |
|-------|---------|
| ข้อมูลกระจาย 10+ ไฟล์ Excel | หาไม่เจอ ตัวเลขไม่ตรงกัน |
| ต้องทำรายงานซ้ำทุกเดือน | เสียเวลา 2-3 วัน/เดือน |
| หัวหน้าถามตัวเลข ตอบทันทีไม่ได้ | ดูไม่เป็นมืออาชีพ |
| เห็นปัญหาช้า (โรคพืชระบาด) | แก้ไขไม่ทัน ความเสียหายขยายวง |

### ✅ Dashboard แก้ปัญหาอย่างไร?

1. **รวมข้อมูลไว้ที่เดียว:** ดูทุกอย่างในหน้าจอเดียว
2. **อัปเดตอัตโนมัติ:** แก้ไขที่ Google Sheets กราฟเปลี่ยนทันที
3. **ตอบคำถามได้ใน 5 วินาที:** เปิดมือถือดูได้ทุกที่
4. **เห็นปัญหาเร็ว:** กราฟพุ่งผิดปกติ = สัญญาณเตือน
5. **ประหยัดเวลา:** ทำครั้งเดียว ใช้ได้ตลอด

### 💡 ตัวอย่างการใช้งานจริง

**สถานการณ์ 1:** นายอำเภอถามว่า "ตำบลไหนปลูกข้าวเยอะสุด?"
- **ก่อน:** เปิด Excel 3 ไฟล์ รวมตัวเลข 20 นาที
- **หลัง:** เปิด Dashboard 5 วินาที ตอบได้ทันที

**สถานการณ์ 2:** เกิดโรคพืชระบาด
- **ก่อน:** รอรายงานจากพื้นที่ 3 วัน
- **หลัง:** เห็นจุดสีแดงบนแผนที่ทันทีที่กรอกข้อมูล

## 🧪 บทที่ 1: เตรียมข้อมูลให้พร้อม

> **"Garbage In, Garbage Out"** ข้อมูลเน่า = Dashboard เน่า

### 📋 Checklist ตารางที่ดี

- [ ] แถวแรก = หัวตาราง (ไม่มีบรรทัดว่างด้านบน)
- [ ] 1 แถว = 1 รายการ (ไม่รวมยอด Total)
- [ ] ไม่มี Merged Cells
- [ ] วันที่เป็น YYYY-MM-DD

### 🛠️ สูตร Google Sheets ที่ต้องรู้

| ปัญหา | สูตร | ผลลัพธ์ |
|-------|------|---------|
| ช่องว่างเกิน | `=TRIM(A2)` | ตัดช่องว่างหน้า-หลัง |
| พิมพ์ผิด | `=SUBSTITUTE(A2,"ต.","ตำบล")` | แทนที่คำ |
| พ.ศ. เป็น ค.ศ. | `=DATE(YEAR(A2)-543,MONTH(A2),DAY(A2))` | แปลงปี |
| หาข้อมูลซ้ำ | `=COUNTIF(A:A,A2)>1` | TRUE = ซ้ำ |
| ดูค่าไม่ซ้ำ | `=UNIQUE(A2:A)` | รายการที่ไม่ซ้ำ |
| แยกข้อความ | `=SPLIT(A2," ")` | แยกด้วยช่องว่าง |

## 🔗 บทที่ 2: เชื่อมต่อและเข้าใจตัวแปร (ฉบับละเอียด)

### 🚀 ขั้นตอนการเชื่อมต่อ Google Sheets กับ Looker Studio

#### Step 1: เปิด Looker Studio
1. เปิด Browser (แนะนำ Chrome)
2. ไปที่ **https://lookerstudio.google.com**
3. Login ด้วย Google Account (Gmail)

#### Step 2: สร้าง Report ใหม่
1. คลิกปุ่ม **"+ Create"** (มุมซ้ายบน)
2. เลือก **"Report"**
3. จะเปิดหน้าเลือก Data Source

#### Step 3: เชื่อมต่อ Google Sheets
1. ในหน้า "Add data to report" → เลือก **Google Sheets**
2. จะแสดงรายการไฟล์ทั้งหมดใน Drive
3. **วิธีหาไฟล์:**
   - พิมพ์ชื่อไฟล์ในช่อง Search
   - หรือเลือกจาก Recent / Shared with me
4. คลิกที่ไฟล์ที่ต้องการ
5. **เลือก Sheet:** ถ้าไฟล์มีหลาย Sheet ให้เลือก Sheet ที่ต้องการ
6. ☑️ ติ๊ก **"Use first row as headers"** (สำคัญมาก!)
7. คลิก **"Add"**
8. คลิก **"Add to report"** เพื่อยืนยัน

> 💡 **Pro Tip:** ตั้งชื่อไฟล์ Google Sheets ให้ชัดเจน เช่น "ข้อมูลเกษตรกร_2567" จะหาง่ายตอนเชื่อมต่อ

### 📏 Dimension vs Metric: หัวใจของการทำกราฟ

นี่คือเรื่องสำคัญที่สุด! ถ้าแยกไม่ออก กราฟจะพังครับ

#### 🟢 Dimension (มิติข้อมูล)
| คุณสมบัติ | รายละเอียด |
|-----------|------------|
| **สีใน Looker** | 🟢 สีเขียว |
| **หน้าที่** | ใช้ **"แบ่งกลุ่ม"** หรือเป็น **แกน X** |
| **คำถามที่ตอบ** | ใคร? อะไร? ที่ไหน? เมื่อไหร่? |
| **ตัวอย่าง** | ชื่อตำบล, ชนิดพืช, วันที่, เพศ, ประเภทเกษตรกร |
| **การคำนวณ** | ไม่นำมาบวกลบคูณหาร |

#### 🔵 Metric (ตัวชี้วัด)
| คุณสมบัติ | รายละเอียด |
|-----------|------------|
| **สีใน Looker** | 🔵 สีฟ้า |
| **หน้าที่** | ใช้ **"คำนวณ"** หรือเป็น **แกน Y** |
| **คำถามที่ตอบ** | เท่าไหร่? เยอะไหม? กี่บาท? กี่คน? |
| **ตัวอย่าง** | จำนวนไร่, จำนวนบาท, จำนวนคน, ผลผลิต (ตัน) |
| **การคำนวณ** | นำมา SUM, AVG, COUNT ได้ |

#### ⚠️ กับดักที่มือใหม่พลาดบ่อย

| ข้อมูล | ดูเหมือน | ควรเป็น | เหตุผล |
|--------|----------|---------|--------|
| รหัสไปรษณีย์ | ตัวเลข | 🟢 Dimension | ไม่เอามาหาค่าเฉลี่ย |
| เบอร์โทรศัพท์ | ตัวเลข | 🟢 Dimension | ไม่เอามาบวกลบ |
| รหัสประชาชน | ตัวเลข | 🟢 Dimension | ไม่มีความหมายทางคณิตศาสตร์ |
| ปี พ.ศ. | ตัวเลข | 🟢 Dimension | ใช้แบ่งกลุ่มตามปี |

### 🔧 วิธีเปลี่ยน Type ของ Field

ถ้า Looker อ่าน Type ผิด (เช่น อ่านวันที่เป็น Text) ให้แก้ไขดังนี้:

1. คลิกที่ **"Resource"** (เมนูด้านบน)
2. เลือก **"Manage added data sources"**
3. คลิก **"Edit"** ที่ Data Source
4. จะเห็นรายการ Field ทั้งหมด
5. คลิกที่ **Type** ของ Field ที่ต้องการแก้ไข
6. เลือก Type ที่ถูกต้อง:
   - **Text** = ข้อความ
   - **Number** = ตัวเลข (คำนวณได้)
   - **Date** = วันที่
   - **Date & Time** = วันที่และเวลา
   - **Geo** = ข้อมูลพิกัด/จังหวัด
7. คลิก **"Done"**

### 📐 Aggregation ที่ต้องรู้ (ฉบับละเอียด)

Aggregation = วิธีที่ Looker Studio คำนวณรวมตัวเลข

| ฟังก์ชัน | ความหมาย | ตัวอย่างการใช้ | ผลลัพธ์ |
|----------|----------|----------------|---------|
| `SUM` | ผลรวม | SUM(พื้นที่_ไร่) | พื้นที่รวมทั้งหมด 1,000 ไร่ |
| `COUNT` | นับแถวทั้งหมด | COUNT(รหัส) | มี 500 แถว |
| `COUNT DISTINCT` | นับไม่ซ้ำ | COUNT DISTINCT(รหัสเกษตรกร) | มีเกษตรกร 450 คน (ไม่นับซ้ำ) |
| `AVG` | ค่าเฉลี่ย | AVG(พื้นที่_ไร่) | พื้นที่เฉลี่ย 5.2 ไร่/คน |
| `MAX` | ค่าสูงสุด | MAX(ผลผลิต) | ผลผลิตมากสุด 2,000 กก. |
| `MIN` | ค่าต่ำสุด | MIN(ผลผลิต) | ผลผลิตน้อยสุด 50 กก. |
| `MEDIAN` | มัธยฐาน | MEDIAN(พื้นที่) | ค่ากลาง (ไม่กระทบ outlier) |

#### 🆚 COUNT vs COUNT DISTINCT

นี่คือความแตกต่างที่สำคัญมาก!

**ตัวอย่างข้อมูล:**
| รหัสเกษตรกร | ชนิดพืช |
|-------------|---------|
| 001 | ข้าว |
| 001 | ส้มโอ |
| 002 | ข้าว |
| 003 | มะพร้าว |

- **COUNT(รหัสเกษตรกร) = 4** (นับทุกแถว)
- **COUNT DISTINCT(รหัสเกษตรกร) = 3** (นับคนไม่ซ้ำ: 001, 002, 003)

> 💡 **ใช้ COUNT DISTINCT เมื่อ:** ต้องการนับจำนวน "คน" หรือ "รายการ" ที่ไม่ซ้ำกัน

### 📊 การเลือก Chart ให้เหมาะกับข้อมูล

#### 1️⃣ Scorecard - แสดงตัวเลขสำคัญ

**ใช้เมื่อ:** ต้องการแสดงตัวเลขสรุปตัวเดียว เช่น ยอดรวม, จำนวนทั้งหมด

**วิธีสร้าง:**
1. คลิก **Add a chart** → **Scorecard**
2. วางบน Canvas
3. ลาก **Metric** (🔵) ใส่ช่อง Metric เช่น `พื้นที่_ไร่`
4. เลือก Aggregation: `SUM`
5. (Optional) เปิด Comparison เพื่อเปรียบเทียบกับช่วงก่อน

**ตัวอย่างผลลัพธ์:**
```
┌─────────────────┐
│   พื้นที่รวม    │
│    1,234       │
│      ไร่        │
│   ▲ +5.2%      │ ← เปรียบเทียบกับเดือนก่อน
└─────────────────┘
```

---

#### 2️⃣ Bar Chart - เปรียบเทียบ

**ใช้เมื่อ:** ต้องการเปรียบเทียบค่าระหว่างหลายกลุ่ม

**วิธีสร้าง:**
1. คลิก **Add a chart** → **Bar chart** (แนวตั้ง หรือ แนวนอน)
2. วางบน Canvas
3. ลาก **Dimension** (🟢) ใส่ช่อง Dimension เช่น `ตำบล`
4. ลาก **Metric** (🔵) ใส่ช่อง Metric เช่น `พื้นที่_ไร่`
5. (Optional) เพิ่ม Sort: พื้นที่มากไปน้อย

**ตัวอย่างผลลัพธ์:**
```
พื้นที่ปลูกแยกตามตำบล
บางระทึก  ████████████████ 450
ศาลายา    ████████████ 320
คลองโยง   █████████ 280
มหาสวัสดิ์ ███████ 200
```

---

#### 3️⃣ Pie Chart - สัดส่วน

**ใช้เมื่อ:** ต้องการแสดงสัดส่วน % ของแต่ละกลุ่ม

**⚠️ ข้อจำกัด:** ไม่ควรเกิน 5-6 ส่วน (ถ้าเกินให้ใช้ Bar Chart แทน)

**วิธีสร้าง:**
1. คลิก **Add a chart** → **Pie chart**
2. วางบน Canvas
3. ลาก **Dimension** (🟢) ใส่ช่อง Dimension เช่น `ชนิดพืช`
4. ลาก **Metric** (🔵) ใส่ช่อง Metric เช่น `พื้นที่_ไร่`

**ตัวอย่างผลลัพธ์:**
```
      สัดส่วนชนิดพืช
         ┌───┐
       ╱ ข้าว ╲
      │  60%   │
       ╲ไม้ผล╱ 25%
         └───┘
          ผัก 15%
```

---

#### 4️⃣ Line Chart - แนวโน้มตามเวลา

**ใช้เมื่อ:** ต้องการดูการเปลี่ยนแปลงตามช่วงเวลา

**วิธีสร้าง:**
1. คลิก **Add a chart** → **Time series** หรือ **Line chart**
2. วางบน Canvas
3. ลาก **Date Dimension** (🟢 วันที่) ใส่ช่อง Dimension
4. ลาก **Metric** (🔵) ใส่ช่อง Metric เช่น `จำนวนเกษตรกร`
5. ตั้งค่า Granularity: Year / Quarter / Month / Week / Day

---

#### 5️⃣ Table - แสดงรายละเอียด

**ใช้เมื่อ:** ต้องการแสดงข้อมูลแบบตาราง รายการละเอียด

**วิธีสร้าง:**
1. คลิก **Add a chart** → **Table**
2. วางบน Canvas
3. ลาก **Dimension** หลายตัวใส่ช่อง Dimension
4. ลาก **Metric** ใส่ช่อง Metric
5. ตั้งค่า Rows per page: 10-25 แถว

### 🎛️ การเพิ่ม Filter และ Control (ละเอียด)

Filter ทำให้ Dashboard โต้ตอบได้ ผู้ใช้เลือกดูเฉพาะข้อมูลที่สนใจ

#### วิธีเพิ่ม Filter
1. คลิก **Add a control** (แถบเครื่องมือด้านบน)
2. เลือกประเภท Control:

| Control | ใช้เมื่อ | ตัวอย่าง |
|---------|---------|----------|
| **Drop-down list** | เลือก 1 ค่าจากรายการ | เลือกตำบล |
| **Fixed-size list** | เลือกได้หลายค่า | เลือกหลายชนิดพืช |
| **Input box** | พิมพ์ค้นหา | ค้นหาชื่อเกษตรกร |
| **Slider** | เลือกช่วงตัวเลข | พื้นที่ 0-100 ไร่ |
| **Checkbox** | เปิด/ปิด | แสดงเฉพาะ Smart Farmer |
| **Date range control** | เลือกช่วงวันที่ | 1 ม.ค. - 31 ธ.ค. 67 |

3. วางบน Canvas (แนะนำ: ด้านบนหรือด้านซ้าย)
4. ลาก **Dimension** ใส่ช่อง **Control field**

#### ⚙️ ตั้งค่า Filter เพิ่มเติม
- **Default selection:** ตั้งค่าเริ่มต้น เช่น แสดง "ทั้งหมด"
- **Search:** เปิดให้พิมพ์ค้นหาในรายการ
- **Metric slider range:** ตั้งค่าต่ำสุด-สูงสุด

---

### 📅 การตั้งค่า Date Range Dimension (สำคัญมาก!)

ถ้าข้อมูลมีคอลัมน์วันที่ และต้องการให้ Filter วันที่ทำงาน:

1. คลิกที่ Chart ใดก็ได้
2. ในแผง Data ด้านขวา หา **"Date Range Dimension"**
3. ลาก Field วันที่ (เช่น `วันที่ขึ้นทะเบียน`) ใส่ช่องนี้
4. ทำซ้ำกับทุก Chart

> ⚠️ **ถ้าไม่ตั้งค่านี้:** Filter วันที่จะไม่ส่งผลต่อ Chart นั้นๆ

---

### 🎨 การปรับแต่ง Style พื้นฐาน

1. คลิกที่ Chart ที่ต้องการแก้ไข
2. คลิกแทบ **"Style"** (ด้านขวา)
3. ปรับแต่งได้:
   - **Title:** ชื่อหัวข้อ Chart
   - **Font:** ฟอนต์และขนาด
   - **Color:** สี Chart
   - **Background:** สีพื้นหลัง
   - **Border:** กรอบ

---

### 💾 การบันทึกและตั้งชื่อ Report

1. คลิกที่ชื่อ "Untitled Report" (มุมซ้ายบน)
2. พิมพ์ชื่อใหม่ เช่น:
   - `[2567] Dashboard พื้นที่ปลูกพืช - อ.พุทธมณฑล`
   - `[Monthly] สรุปผลงานส่งเสริมการเกษตร`
3. Report จะ Auto-save ทุกครั้งที่แก้ไข

## 🔥 บทที่ 4: เทคนิคขั้นสูง

### 1. Calculated Fields (สูตรคำนวณ) 🧮

**ผลผลิตต่อไร่:**
```sql
SUM(ผลผลิต_กก) / SUM(พื้นที่_ไร่)
```

**ตัดเกรดเกษตรกร:**
```sql
CASE
    WHEN พื้นที่_ไร่ >= 50 THEN "รายใหญ่"
    WHEN พื้นที่_ไร่ >= 20 THEN "รายกลาง"
    ELSE "รายย่อย"
END
```

### 2. Date Functions �

| ฟังก์ชัน | ใช้งาน |
|----------|--------|
| `YEAR(date)` | ดึงปี |
| `MONTH(date)` | ดึงเดือน |
| `WEEK(date)` | ดึงสัปดาห์ |
| `DATE_DIFF(date1,date2)` | นับจำนวนวันระหว่าง 2 วัน |

### 3. Drill Down 🔽
ตั้งค่าให้กราฟกดลงลึกได้: จังหวัด → อำเภอ → ตำบล

### 4. Data Blending 🔀
รวมข้อมูลจาก 2 ไฟล์ด้วย Key ที่เหมือนกัน (เช่น รหัสตำบล)

## 📖 ภาคผนวก: คำศัพท์

| คำศัพท์ | ความหมาย |
|---------|----------|
| Dimension | มิติข้อมูล (แบ่งกลุ่ม) |

| Aggregation | การคำนวณรวม (SUM, AVG) |
| Filter | ตัวกรองข้อมูล |
| Control | เครื่องมือโต้ตอบ |
| Calculated Field | สูตรที่สร้างเอง |
